## 请求<a name="section44958995161021"></a>

-   参数说明

    无

-   请求样例

    查询Zone ID为“ff8080825b8fc86c015b94bc6f8712c3”的内网域名的标签。

    ```
    GET https://{DNS_Endpoint}/v2/{project_id}/DNS-private_zone/ff8080825b8fc86c015b94bc6f8712c3/tags
    ```
