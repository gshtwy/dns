# 查询域名信息<a name="ZH-CN_TOPIC_0156607176"></a>

## 功能介绍<a name="zh-cn_topic_0124033548_zh-cn_topic_0037134402"></a>

## URI<a name="zh-cn_topic_0124033548_section44955787"></a>
