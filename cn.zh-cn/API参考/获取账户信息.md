# 获取账户信息<a name="ZH-CN_TOPIC_0156607172"></a>

## 功能介绍<a name="zh-cn_topic_0124033545_section42452169"></a>

## URI<a name="zh-cn_topic_0124033545_section46525208"></a>

GET /v1/account

## 请求<a name="zh-cn_topic_0124033545_section16073696"></a>
